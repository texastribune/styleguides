Stylesheet

This is the style guide for TribTalk, a publication of the Texas Tribune. TribTalk's styles are written in Sass. TribTalk uses the Bourbon mixin library for some addons, CSS3 elements, functions, and helpers. It also uses Neat for its grid system and responsive layout. Below you'll find more details about the elements that give TribTalk its unique look.

Sass/CSS

Sass styles are written in .sass. All Sass files are imported into the project's main Sass file, app.sass, and then Grunt is used to compile it to app.css and minify it. Compiled CSS for TribTalk is not checked into the GitHub repo.

JavaScript

JavaScript scripts that need to be loaded include jQuery, Backbone, Underscore, Moment, Modernizr, jQuery Cookie, jQuery scrollTo, and XscrollY. Scripts should be loaded from a cdn or AWS. Backbone is used for adding comments and giving respect to comments. TribTalk uses one jQuery plugin, simple modal. JavaScript is also used to toggle the nav menu and to implement infinite scroll.

Grid

Neat provides the foundation for the grid system for TribTalk.

Header

For the site header, .site-header is used to establish the height, width, and fixed position of the header. A div with class .wrapper is then placed inside the header.

                
                  <header class="site-header">
                    <div class="wrapper">
                    </div>
                  </header>

Main Content

The main content is placed within a div with class .wrapper to center it and give it a max width. The span-columns mixin is used to set the number of columns an element should span. The omega mixin is used to remove elements' gutter margin.

                
                  <div class="wrapper">
                  </div>

Navigation

Main Menu

Main site navigation is activated with the hamburger icon. jQuery is used to add classes that toggle the menu open and closed when clicked.

The menu includes About, Authors, Submit, Paid Placement, Donate, and a link to the main Texas Tribune site.

Images

It's preferred that img tags always have alt tags to fall back on. Alt tags should communicate as close to the same information as the image as possible.

Homepage Slider

There are three images slated for the slider at the top of the homepage. These images have a class of .lede_art. They are placed inside a div with a class of .flexslider, then a ul with class of .slides, then an li with class .flex-active-slide. These images are inside links that go to the main article detail page.

                
                  <div class="flexslider">
                    <ul class="slides">
                      <li class="flex-active-slide">
                        <img class="lede_art" src="">
                      </li>
                    </ul>
                  </div>

Homepage

Images on the homepage below the slider are wrapped in divs with a class of .media. They are then images with a class of .lede_art.

                
                  <div class="media">
                    <img class="lede_art" src="">
                  </div>

Main Article Detail Page

Images on the main article detail page have a class of .lede_art.

                
                  <img class="lede_art" src="" data-ar="">

Authors Page

Images on the authors page are placed inside a body with class .author-list, a div with class .author-summary, and then img. These images are placed inside links to each author's detail page.

                
                  <body class="author-list">
                    <div class="author-summary">
                      <a href="#">
                        <img>
                        </img>
                      </a>
                    </div>
                  </body>

Bylines & Credits

Bylines

Bylines are placed inside of a paragraph with a class of byline and a span with a class of authors. The copy for bylines is "By FirstName LastName, Month Day, Year". Authors' names should be links going to their author detail page.

                
                  <p class="byline">
                    <span class="authors">
                      By <a href="#">FirstName LastName</a>, Month Day, Year
                    </span>
                  </p>

Image Credits

Photo credits appear as italicized captions beneath the appropriate images. Use an outer paragraph with class caption, and then place a span with a class of credit around the credit. The copy should be "Photo by FirstName LastName".

                
                  <p class="caption">
                    <span class="credit">
                      Photo by FirstName LastName
                    </span>
                  </p>

Paragraphs

Homepage - article body

Paragraphs inside .article_body have a font size of 1.1em, line height of 1.4, color #282828, bottom margin of .75em, top margin of 0, and font weight of 200. This style is found on TribTalk's homepage.

              
                <div class=".article_body">
                  <p>
                    ...
                  </p>
                </div>

Main article detail

Paragraphs inside .article_detail and .main article are styled with font size of 1.1em, line height of 1.4, and color #282828. You'll find this style on individual article pages.

                
                  <body class="article_detail">
                    <div class="main">
                      <article>
                        <p>
                          ...
                        </p>
                      </article>
                    </div>
                  </body>

About, Submit & Paid Placement pages

Paragraphs here have body class .page and div class .main. They're styled the same way as main article detail pages.

                
                  <body class="page">
                    <div class="main">
                      <p>
                        ...
                      </p>
                    </div>
                  </body>

Lists

Lists can be found on the submit page. Lists are styled with bullets delineating each line to the left of the text.

              
                <body class="page">
                  <div class="main">
                    <ul>
                      <li>
                        ...
                      </li>
                    </ul>
                  </div>
                </body>

Comment Form

The comment form appears on main article detail pages. The comment form heading is an h2. Apply the following styles for comment forms.

              
                <div class="comments-container">
                  <div class="comment-form-container">
                  </div>
                </div>

Icons

Social Media

Font Awesome is used for icons and is loaded from a cdn above the main CSS. Facebook and Twitter social media icons are included in the header and footer. On main article detail pages, icons appear both to the right and below the article. Facebook, Twitter, and Linkedin are included here, as well as icons for sharing via email and a link to joining the Tribune. The hover color is avocado. Icons that are links should have target="_blank" to ensure that they open in new windows.

              
                <i class="fa fa-{name}">
                </i>
