Stylesheet

Welcome to the style guide for TribTalk, a publication of the Texas Tribune. TribTalk's styles are written originally in Sass. TribTalk uses the Bourbon mixin library for some addons, CSS3 elements, functions, and helpers. It also uses Neat for its grid system and responsive layout. Below you'll find more details about the elements that give TribTalk its unique look.

Branding

Grid

Neat provides the foundation for the grid system for TribTalk.

Header

For the site header, .site-header is used to establish the height, width, and fixed position of the header. A div with class .wrapper is then placed inside the header, and here it specifically utilizes the $pad mixin from Neat to set the padding around the h1 element.

Main Content

The main content is placed within a div with class .wrapper to center it and give it a max width. The span-columns mixin is used to set the number of columns an element should span. The omega mixin is used to remove elements' gutter margin.

Paragraphs

Homepage - article body

Paragraphs inside .article_body have a font size of 1.1em, line height of 1.4, color #282828, bottom margin of .75em, top margin of 0, and font weight of 200. This style is found on TribTalk's homepage.

Main article detail

Paragraphs inside .article_detail and .main_article are styled with font size of 1.1em, line height of 1.4, and color #282828. You'll find this style on individual article pages.

About, Submit & Paid Placement pages

Paragraphs here have body class .page and div class .main. They're styled the same way as main article detail pages.
